# Vantage-Gadgets-Executive-Profitability-Dashboard
This project is a high-fidelity E-commerce Executive Dashboard built in Microsoft Excel. It transforms raw order, product, and marketing data into a strategic tool for identifying "Money Pit" SKUs and optimizing advertising spend.

The dashboard provides a "single pane of glass" view for decision-makers to analyze the journey from Gross Sales to Net Profit and take immediate action on underperforming products.

üöÄ Key Features

Waterfall Financial Flow: Visualizes the impact of refunds, COGS, shipping, and ad spend on the final Net Profit ($97,445).

Automated Action Engine: A custom logic-driven table that categorizes SKUs as "Winners" to scale or "Money Pits" to stop, based on net margin performance.

Interactive Slicer UI: A modern, Indigo-themed interface allowing users to filter the entire dashboard by recommended action with one click.

Executive KPI Cards: High-contrast cards highlighting critical metrics like Store ROAS (128.93x) and Monthly Ad Waste ($18,928).

Dynamic Insight Banner: A smart text engine that calculates and displays the exact monthly savings achieved by following the dashboard's recommendations.
